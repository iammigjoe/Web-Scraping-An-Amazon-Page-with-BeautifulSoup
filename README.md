# SimpleWeb-Scraping-Project-Of-An-Amazon-Page-with-BeautifulSoup
I built a web scraping script using BeautifulSoup in Jupyter Notebook to extract product information from an Amazon page and stored the collected data in a CSV file for further analysis.

French version
Web Scraping d’une page Amazon
J’ai développé un script de web scraping avec BeautifulSoup dans Jupyter Notebook pour extraire des informations produits d’une page Amazon et j’ai enregistré les données collectées dans un fichier CSV pour analyse ultérieure.

--------------

Project: Web Scraping an Amazon Page
	•	Description: Developed a web scraping script to extract product information (titles, prices, ratings, etc.) from an Amazon page and save it into a CSV file for further analysis.
	•	Technologies & Libraries: Python, Jupyter Notebook, BeautifulSoup, Requests, Pandas, CSV module
	•	Skills Applied: Data extraction, HTML parsing, data cleaning, CSV data storage
	•	Outcome: Successfully collected structured product data for analysis and visualization


Projet : Web Scraping d’une page Amazon
	•	Description : Développement d’un script de web scraping pour extraire des informations produits (titres, prix, évaluations, etc.) d’une page Amazon et les enregistrer dans un fichier CSV pour analyse ultérieure.
	•	Technologies & Bibliothèques : Python, Jupyter Notebook, BeautifulSoup, Requests, Pandas, module CSV
	•	Compétences mises en œuvre : Extraction de données, parsing HTML, nettoyage des données, stockage dans un fichier CSV
	•	Résultat : Données produits structurées collectées avec succès pour analyse et visualisation
